Nesta atividade usamos o site github.com para criar um repositório para poder clonar na nossa máquina, usando o programa Git Bash, posteriormente foi criada uma aplicação Java simples no programa Intelij, com uma classe Main, que contém o método principal que imprime a mensagem "Avaliação Git com Java", foi versionado e enviado para o Git Hub. Depois foi criado uma nova branch no Intelij chamada funcoes que foi criado nela a classe Calculadora para realizar operações de soma, multiplicação e divisão, após foi mesclado a branch funcoes com a main. 
A atividade teve como finalidade praticar a estrutura básica de um programa em Java, aplicação de conceitos de orientação a objetos e preparação do código para controle de versão com Git.
